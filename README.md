# django-playground

## python pip

* Install: <br>
   ```python3 -m pip install --user --upgrade pip```
* Check version: <br>
  ```python3 -m pip --version```

## Virtual environment

* Install: <br>
   ```python3 -m pip install --user virtualenv```
* Create: <br>
   ```python3 -m venv env```
* Activate: <br>
   ```source env/bin/activate```
* Deactivate: <br>
   ```deactivate```
